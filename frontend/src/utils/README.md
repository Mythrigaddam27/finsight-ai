# Utils

Utility functions and helpers for common operations.

## Purpose

Houses pure functions and utility helpers used throughout the application:

- Date/time formatting and manipulation
- Currency and number formatting
- String utilities
- Array/object manipulation helpers
- Validation functions
- Constants and enums
- URL/route helpers

## Guidelines

- Functions should be pure (no side effects)
- Functions should be well-documented
- Group related utilities together
- Avoid coupling utilities to specific business logic
- Unit test utility functions

## Usage

Utilities are imported and used throughout components, hooks, and services to avoid duplicating common functionality.
