# Expenses Components

Feature-specific components for expense management functionality.

## Purpose

Contains all UI components related to expense tracking and management:

- Expense forms and inputs
- Expense tables and lists
- Expense cards and items
- Category selectors
- Modal dialogs (create, edit, delete)
- Filter and search components
- Receipt upload/display components

## Usage

These components are used together to build the expense management feature pages and workflows. They handle expense creation, editing, viewing, and filtering operations.
